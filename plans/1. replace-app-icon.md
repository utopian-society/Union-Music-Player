# Plan: Replace App Launcher Icon

## Goal
Replace the current music note vector icon with the new speaker icon from `photo_2026-03-30_10-24-50.jpg`

## Steps

### 1. Prepare the Image
- Convert JPG to PNG format
- Generate multiple sizes for different screen densities:
  - xxxhdpi: 432x432 px (full icon), 324x324 px (foreground)
  - xxhdpi: 324x324 px (full icon), 243x243 px (foreground)
  - xhdpi: 216x216 px (full icon), 162x162 px (foreground)
  - hdpi: 162x162 px (full icon), 121.5x121.5 px (foreground)
  - mdpi: 108x108 px (full icon), 81x81 px (foreground)

### 2. Place PNG Files
- Place foreground PNG in each density folder:
  - `app/src/main/res/mipmap-xxxhdpi/ic_launcher_foreground.png`
  - `app/src/main/res/mipmap-xxhdpi/ic_launcher_foreground.png`
  - `app/src/main/res/mipmap-xhdpi/ic_launcher_foreground.png`
  - `app/src/main/res/mipmap-hdpi/ic_launcher_foreground.png`
  - `app/src/main/res/mipmap-mdpi/ic_launcher_foreground.png`

### 3. Update Adaptive Icon XML
- Modify `ic_launcher.xml` to reference PNG instead of vector drawable

### 4. Build and Test
- Run `./gradlew clean assembleDebug`
- Install on device/emulator to verify

## Implementation Details

The new icon features:
- Red speaker with two black circles (drivers)
- Colorful环绕 effect (green, blue, white)
- Green background (matching app theme #4CAF50)
