音乐播放器UI改造方案（Material 3 + Jetpack Compose）

基于你的需求，我设计了以下UI布局方案：

一、整体架构

1. 应用结构

MainActivity
├── MusicApp (可组合函数)
│   ├── MusicTheme (自定义主题)
│   ├── Scaffold (主框架)
│   │   ├── TopAppBar (顶部栏 - 移除账户图标)
│   │   ├── Navigation (页面导航)
│   │   │   ├── LibraryScreen (资料库页)
│   │   │   └── MoreScreen (更多页)
│   │   └── MiniPlayer (底部迷你播放器)
│   └── BottomNavigation (底部导航栏)

2. 页面规划

- 资料库页 (Library)：展示专辑卡片网格
- 更多页 (More)：设置、播放历史、偏好等

二、颜色主题 (Material 3)

// 定义颜色主题
private val GreenPrimary = Color(0xFF4CAF50)      // 绿色主色
private val GreenSecondary = Color(0xFF8BC34A)    // 绿色辅助色
private val YellowAccent = Color(0xFFFFC107)      // 黄色强调色
private val TranslucentWhite = Color(0x80FFFFFF)  // 半透明白色

val MusicColorScheme = lightColorScheme(
    primary = GreenPrimary,
    secondary = GreenSecondary,
    tertiary = YellowAccent,
    surface = TranslucentWhite.copy(alpha = 0.9f),
    onSurface = Color.Black
)

3. 资料库页面 (专辑卡片网格)

@OptIn(ExperimentalLayoutApi::class)
LibraryScreen {
    // 精选推荐区域
    Text("精选推荐", style = MaterialTheme.typography.headlineMedium)
    
    // 专辑卡片网格
    LazyVerticalGrid(
        columns = AdaptiveGridItem.Adaptive(minSize = 180.dp)
    ) {
        items(albums) { album ->
            AlbumCard(album)
        }
    }
}

// 专辑卡片组件
@Composable
fun AlbumCard(album: Album) {
    Card(
        modifier = Modifier
            .padding(8.dp)
            .fillMaxWidth()
            .aspectRatio(1f),
        colors = CardDefaults.cardColors(
            containerColor = MaterialTheme.colorScheme.surfaceVariant
        ),
        elevation = CardDefaults.cardElevation(defaultElevation = 4.dp)
    ) {
        Column {
            // 专辑封面
            Image(
                painter = rememberAsyncImagePainter(album.coverUrl),
                contentDescription = album.title,
                modifier = Modifier
                    .fillMaxWidth()
                    .weight(1f)
                    .clip(RoundedCornerShape(topStart = 12.dp, topEnd = 12.dp))
            )
            
            // 专辑信息
            Column(modifier = Modifier.padding(12.dp)) {
                Text(
                    text = album.title,
                    style = MaterialTheme.typography.titleMedium,
                    maxLines = 1,
                    overflow = TextOverflow.Ellipsis
                )
                Text(
                    text = album.artist,
                    style = MaterialTheme.typography.bodySmall,
                    color = MaterialTheme.colorScheme.onSurfaceVariant
                )
            }
        }
    }
}

4. 底部迷你播放器

MiniPlayer(
    modifier = Modifier
        .fillMaxWidth()
        .height(70.dp)
        .background(
            Brush.verticalGradient(
                colors = listOf(
                    TranslucentWhite.copy(alpha = 0.95f),
                    TranslucentWhite.copy(alpha = 0.9f)
                )
            )
        )
        .blur(radius = 12.dp)
) {
    Row(
        modifier = Modifier
            .fillMaxSize()
            .padding(horizontal = 16.dp),
        verticalAlignment = Alignment.CenterVertically
    ) {
        // 歌曲信息
        Column(
            modifier = Modifier.weight(1f)
        ) {
            Text(
                text = currentSong.title,
                style = MaterialTheme.typography.bodyLarge,
                maxLines = 1,
                overflow = TextOverflow.Ellipsis
            )
            Text(
                text = currentSong.artist,
                style = MaterialTheme.typography.bodySmall,
                color = MaterialTheme.colorScheme.onSurfaceVariant
            )
        }
        
        // 播放控制按钮
        Row(
            horizontalArrangement = Arrangement.spacedBy(16.dp)
        ) {
            IconButton(onClick = { /* 上一首 */ }) {
                Icon(Icons.Filled.SkipPrevious, null, tint = GreenPrimary)
            }
            
            IconButton(
                onClick = { /* 播放/暂停 */ },
                modifier = Modifier.size(40.dp)
            ) {
                Icon(
                    if (isPlaying) Icons.Filled.Pause else Icons.Filled.PlayArrow,
                    null,
                    tint = YellowAccent
                )
            }
            
            IconButton(onClick = { /* 下一首 */ }) {
                Icon(Icons.Filled.SkipNext, null, tint = GreenPrimary)
            }
        }
    }
}

四、状态管理

// ViewModel
class MusicViewModel : ViewModel() {
    // 当前播放状态
    var currentSong by mutableStateOf(Song())
    var isPlaying by mutableStateOf(false)
    
    // 专辑列表
    val albums by mutableStateOf(listOf<Album>())
    
    // 当前页面
    var currentScreen by mutableStateOf(Screen.Library)
}

// 屏幕枚举
enum class Screen {
    Library, More
}

五、实现建议

1. 依赖项：

implementation "androidx.compose.material3:material3:1.1.0"
implementation "androidx.compose.ui:ui:1.5.0"
implementation "androidx.navigation:navigation-compose:2.6.0"

2. 性能优化：
   - 使用 
"remember" 缓存计算昂贵的操作
   - 对图片使用 
"Coil" 或 
"Glide" 进行异步加载和缓存
   - 使用 
"Lazy" 组件处理长列表
3. 交互效果：
   - 为按钮添加 
"animatedContentSize" 动画
   - 使用 
"Modifier.hoverable" 添加悬停效果
   - 实现卡片点击的波纹效果
