# Apple Music 风格 Library 屏幕实现方案

## 概述
实现一个 Apple Music 风格的"资料库"屏幕，具有现代化的视觉效果和毛玻璃模糊效果。

## 视觉属性

### Bottom Navigation Bar
- **圆角**: 12dp (`RoundedCornerShape(12.dp)`)
- **背景**: `Color.White.copy(alpha = 0.9f)` (90% 不透明白色)
- **模糊效果**: `BackdropFilter` with `BlurEffect(8f, 8f)`
- **位置**: 屏幕底部固定
- **高度**: 80dp

### Mini Player
- **圆角**: 16dp (`RoundedCornerShape(16.dp)`)
- **背景**: `Color.White.copy(alpha = 0.85f)` (85% 不透明白色)
- **模糊效果**: `BackdropFilter` with `BlurEffect(8f, 8f)`
- **位置**: Bottom Navigation Bar 上方
- **高度**: 70dp
- **内容**: 专辑封面 (60dp), 歌曲标题，艺术家名，播放控制

### 核心 UI 元素
- **状态栏**: 模拟显示时间 (9:41) 和系统图标
- **Header**: "Library" 标题 (34sp, Bold)
- **专辑行**: 水平滚动，160dp x 160dp 封面，8dp 圆角
- **导航网格**: 2 列布局，图标 + 文字
- **最近添加**: 60dp 封面 + 歌曲信息 + 播放控制

## 技术实现

### 依赖
- Jetpack Compose 1.7.8+
- Material Design 3
- Coil (图片加载)

### 关键 Modifier
```kotlin
// 圆角
.clip(RoundedCornerShape(12.dp))

// 模糊背景 (需要 Android 12+)
.graphicsLayer {
    renderEffect = RenderEffect.createBlurEffect(8f, 8f, Shader.TileMode.CLAMP)
}

// 半透明背景
.background(Color.White.copy(alpha = 0.9f))
```

### 最小屏幕宽度
- 360dp (标准手机宽度)
