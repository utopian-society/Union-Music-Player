🎨 Compose组件树结构

// 主屏幕整合
@Composable
fun AppScreen() {
    val playerState = rememberPlayerState()
    
    Box(modifier = Modifier.fillMaxSize()) {
        // 1. 你的应用主内容
        YourAppContent()
        
        // 2. 播放器系统
        PlayerSystem(
            state = playerState,
            modifier = Modifier.align(Alignment.BottomCenter)
        )
    }
}

// 播放器系统
@Composable
fun PlayerSystem(
    state: PlayerState,
    modifier: Modifier = Modifier
) {
    AnimatedContent(
        targetState = state.isExpanded,
        transitionSpec = { /* 扩展动画 */ }
    ) { isExpanded ->
        if (isExpanded) {
            FullScreenPlayer(state = state)
        } else {
            MiniPlayer(state = state)
        }
    }
}

## 项目结构
创建以下Compose文件结构：

## 技术要求
1. **使用Compose最新稳定版**
2. **状态管理**: 使用`ViewModel`和`mutableStateOf`
3. **手势处理**: 实现水平滑动切换页面，垂直下拉收起
4. **动画**: 使用`AnimatedContent`和`updateTransition`实现平滑过渡
5. **响应式设计**: 适配不同屏幕尺寸

## 交互流程
1. **初始状态**: 显示迷你播放器在屏幕底部
2. **点击展开**: 点击任意位置展开为全屏播放器
3. **页面切换**: 水平滑动切换"专辑封面"和"歌词"页面
4. **收起方式**:
   - 点击顶部返回按钮
   - 向下滑动超过30%
   - 点击底部空白区域
